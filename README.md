# 📖 Project Overview

As part of the **ApexPlanet 30-Day Data Analytics Internship**, this project demonstrates the complete visualization workflow using Python and Power BI.

The objective was to transform raw **Superstore Sales** data into meaningful business insights through static charts, interactive visualizations, and an executive-level Power BI dashboard.

This project combines **Exploratory Data Analysis (EDA)**, **Data Visualization**, and **Business Intelligence** techniques to support data-driven decision-making.

# ▶️ Getting Started

### Clone the Repository

```bash
git clone https://github.com/Hariomdubey01/ApexPlanet-Task3.git
```

### Install Dependencies

```bash
pip install pandas numpy matplotlib seaborn plotly kaleido
```

### Run the Notebook

```bash
jupyter notebook notebooks/03_task.ipynb
```

### Open the Power BI Dashboard

Open:

```
dashboards/superstore_dashboard.pbix
```

using **Microsoft Power BI Desktop**.
